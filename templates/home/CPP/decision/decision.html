Decision Making



If Statment

Use the if statement to specify a block of C++ code to be executed if a condition is true.

Example

                
                      if (20 > 18) {
    cout << "20 is greater than 18";
    }

Else Statment

Use the else statement to specify a block of code to be executed if the condition is false.

Example

                
                  int time = 20;
   if (time < 18) {
    cout << "Good day.";
   } else {
     cout << "Good evening.";
  }
   // Outputs "Good evening."

Switch Statment

Use the switch statement to select one of many code blocks to be executed.

Example

                
                  int day = 4;
switch (day) {
     case 1:
   cout << "Monday";
    break;
  case 2:
    cout << "Tuesday";
    break;
  case 3:
    cout << "Wednesday";
    break;
  case 4:
    cout << "Thursday";
    break;
  case 5:
    cout << "Friday";
   break;
  case 6:
    cout << "Saturday";
   break;
  case 7:
    cout << "Sunday";
    break;
 }
// Outputs "Thursday" (day 4)
