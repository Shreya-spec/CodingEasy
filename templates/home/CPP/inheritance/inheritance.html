Inheritance

In C++, it is possible to inherit attributes and methods from one class to another. We group the "inheritance concept" into two categories:

  • Derived class

    A class that inherits from another class.

  • Base class

    A class that is inherited by another class.

To inherit from a class, use the : symbol.

Implementation of inheritance

For creating a sub-class that is inherited from the base class we have to follow the below syntax. 

          
            class SubClass : access specifier BaseClass {
  // code to be executed
}

Here, subclass_name is the name of the subclass, access_mode is the mode in which you want to inherit the subclass for example public, private, etc. and base_class_name is the name of the base class from which you want to inherit the subclass. Note: A derived class doesnâ€™t inherit access to private data members. However, it does inherit a full parent object, which contains any private members which that class declares.

Example

                               
                               //single inheritance example
                                class A {
                                public:
                                void Car() {
                                cout << "I am  Car\n";
                                }
                                };
                                class B : public A {
                                public:
                                void Bus() {
                                cout << "I am Bus\n";
                                }
                                };
                                int main() {
                                B b;
                                b.Car();
                                return 0;
                                }

                                //output
                                I am  Car

Multiple inheritance

Multiple Inheritance is a feature of C++ where a class can inherit from more than one class. i.e one subclass is inherited from more than one base class.

Example

                               
                               //multiple inheritance example
                                class A {
                                public:
                                void Car() {
                                cout << "I am  Car\n";
                                }
                                };
                                class B : public A {
                                public:
                                void Bus() {
                                cout << "I am Bus\n";
                                }
                                };
                                class C : public A {
                                public:
                                void Bike() {
                                cout << "I am Bike\n";
                                }
                                };
                                class D : public B, public C {
                                public:
                                void Car() {
                                cout << "I am Car\n";
                                }
                                };
                                int main() {
                                D d;
                                d.Car();
                                d.Bus();
                                d.Bike();
                                return 0;
                                }

                                //output
                                I am Car
                                I am Bus
                                I am Bike

Multilevel inheritance

In this type of inheritance, a derived class is created from another derived class.

Example

                               
                               //multilevel inheritance
                                class A {
                                public:
                                void Car() {
                                cout << "I am  Car\n";
                                }
                                };
                                class B : public A {
                                public:
                                void Bus() {
                                cout << "I am Bus\n";
                                }
                                };
                                class C : public B {
                                public:
                                void Bike() {
                                cout << "I am Bike\n";
                                }
                                };
                                class D : public C {
                                public:
                                void Car() {
                                cout << "I am Car\n";
                                }
                                };
                                int main() {
                                D d;
                                d.Car();
                                d.Bus();
                                d.Bike();
                                return 0;
                                }

                                //output
                                I am Car
                                I am Bus
                                I am Bike

Hierarchical inheritance

In this type of inheritance, more than one subclass is inherited from a single base class. i.e. more than one derived class is created from a single base class.

Example

                               
                               //hierarchical inheritance
                                class A {
                                public:
                                void Car() {
                                cout << "I am  Car\n";
                                }
                                };
                                class B : public A {
                                public:
                                void Bus() {
                                cout << "I am Bus\n";
                                }
                                };
                                class C : public A {
                                public:
                                void Bike() {
                                cout << "I am Bike\n";
                                }
                                };

                                int main() {
                                B b;
                                b.Car();
                                b.Bus();
                                C c;
                                c.Car();
                                c.Bike();

                                //output
                                I am  Car
                                I am Bus
                                I am  Car
                                I am Bike
