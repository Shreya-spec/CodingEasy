Advance Git Commands

git switch
// If you have more than one branch in your repo, you can switch between them using the git switch command.

git switch {branch-name}

git diff
// When you have made some changes to your code and want to see exactly what you have changed, you can use the git diff command.

git diff

git reset
// When you want to revert your changes, you can use the git reset command. It is a powerful command that is used to undo local changes to the state of a Git repo.

git reset

git show
// When you want to see the history of a file, you can use the git show command. It describes the HEAD commit by default

git show

git merge
// When you want to merge two branches, you can use the git merge command. It merges the changes from one branch into another.

git merge {branch-name}

git remote
// When you want to see the list of remotes, you can use the git remote command. It lists the remotes in your repo.

git remote

git reset
// When you want to revert your changes, you can use the git reset command. It is a powerful command that is used to undo local changes to the state of a Git repo.

git reset

git checkout
// When you want to checkout a branch, you can use the git checkout command. It switches to the specified branch.

git checkout {branch-name}

git stash
// When you want to stash your changes, you can use the git stash command. It saves your changes in a temporary stash (doesnt commit but keeps track).

git stash
